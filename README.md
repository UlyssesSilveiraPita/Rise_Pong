# 🏓 Rise Pong

Projeto retrô desenvolvido em Unity inspirado no clássico Pong.

O objetivo do projeto foi estudar e aplicar conceitos fundamentais de gameplay programming, física 2D e arquitetura modular utilizando C#.

---

## 🎮 Gameplay

Controle sua raquete e rebata a bola contra o oponente em partidas rápidas inspiradas nos arcades clássicos.

---

## ⚡ Mecânicas Implementadas

- Movimento das raquetes
- Física da bola
- Sistema de colisão
- Controle de input
- IA básica para oponente
- Sistema de pontuação
- Reinício de partida

---

## 🛠️ Tecnologias Utilizadas

- Unity
- C#
- Física 2D
- Gameplay Systems
- Arquitetura Modular

---

## 🎯 Objetivos do Projeto

Este projeto foi desenvolvido para estudo de:

- gameplay programming;
- lógica de jogos;
- controle de input;
- organização de código;
- sistemas modulares em Unity.
